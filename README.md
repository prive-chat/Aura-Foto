# Aura Studio | AI Artistic Portrait Studio

Aura Studio is a high-end, cinematic web application designed for creating professional-grade artistic portraits using Artificial Intelligence. Powered by **Google Gemini**, it combines cutting-edge AI generation with a refined, immersive user interface featuring **Glassmorphism 2.0**.

## ✨ Key Features

- **Cinematic AI Generation**: High-fidelity artistic portraits with advanced styling.
- **Glassmorphism 2.0 UI**: A premium, dark-themed immersive experience.
- **PWA Ready**: Installable on iOS, Android, and Desktop as a native application.
- **Secure Authentication**: Integrated with Supabase for user data protection.
- **Admin Terminal**: Full control over system stats, user quotas, and global image library.
- **Character Library**: Quick-start tokens to instantly create consistent personas.
- **Advanced Editor**: Refine and manipulate generated works with precision.

## 🛠 Tech Stack

- **Frontend**: React 19, TypeScript, Vite.
- **Styling**: Tailwind CSS v4, Motion (Framer Motion).
- **IA Engine**: @google/genai (Gemini 3.1 Pro/Flash).
- **Backend/Auth**: Supabase.
- **PWA**: Service Workers, Web Manifest.

## 🚀 Getting Started

### Prerequisites

- Node.js 20+
- npm or yarn

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/your-username/aura-studio.git
   cd aura-studio
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Set up environment variables:
   Create a `.env` file based on `.env.example`.

4. Start development server:
   ```bash
   npm run dev
   ```

## 📱 PWA Support

Aura Studio is a Progressive Web App. You can install it on your device:
- **Desktop**: Click the "Download" icon in the URL bar.
- **Mobile**: Use "Add to Home Screen" from your browser menu.
